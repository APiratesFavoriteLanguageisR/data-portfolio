# NYC Taxi ETL Pipeline

## Overview
This project builds a modular ETL pipeline using NYC taxi trip data.

The pipeline extracts data from BigQuery, applies cleaning and validation transformations, and prepares an analysis-ready dataset.

## Architecture
Extract → Transform → Load pipeline with modular transformation stages.

## Technologies
Python
Pandas
BigQuery
YAML configuration
