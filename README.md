# Projects

## NYC Taxi ETL Pipeline

A modular ETL pipeline that extracts NYC taxi trip data from BigQuery, performs data cleaning and validation transformations, and prepares an analysis ready dataset.

Key features

* Config driven pipeline architecture
* Modular transformation stages
* Data validation and cleaning
* Feature engineering for analytical use

Technologies
Python, Pandas, BigQuery, YAML

Technologies
Python, LLM APIs, embeddings
